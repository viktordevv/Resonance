Getting Started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /css — folder with css files.
  2. /images — folder with image files.
  3. /videos — folder with video files.
  4. /js — folder with Javascript files.
  5. /php — folder with PHP files.
  6. /webfonts — folder with font files.

  7. corporate-multi-page.html — page with the list of the template pages.
  8. bold-*.html — Bold design concept demo pages.
  9. brutalist-*.html — Brutalist design concept demo pages.
  10. corporate-*.html — Corporate design concept demo pages.
  11. elegant-*.html — Elegant design concept demo pages.
  12. fancy-*.html — Fancy design concept demo pages.
  13. gradient-*.html — Gradient design concept demo pages.
  14. main-*.html — Main design concept demo pages.
  15. modern-*.html — Modern design concept demo pages.
  16. slick-*.html — Slick design concept demo pages.
  17. strong-*.html — Strong design concept demo pages.
  18. *-dark.html — dark version for each page.

  19. contact_me.php — file with contact form settings.
  20. contact_me_smtp.php — file with contact form settings (SMTP method).

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients (for example, FileZilla or Total Commander).

Note: for the correct operation of all functions of the template, including AJAX methods of Contact form, you must upload the template to web-server.

    Contacts Form

    You can to use two different contact form sending methods: PHP Mail function and SMTP.

    PHP Mail Function Method

    To configure the contact form, open file php/contact_me.php and enter your email in this line (~line 4): 

                            $to_Email = "your-email@domain.com";

    SMTP Method

    If your web hosting doesn't support PHP Mail function, you can use SMTP method. To configure the contact form with SMTP method, open file php/contact_me_smtp.php and enter your email, host, username and password in this code fragment (~line 4-11): 

                                $to_Email       = "support@bestlooker.pro"; // Replace with recipient email address
                            $subject        = 'Message from my website'; // Subject line for emails
                            
                            $host           = "smtp.mail.yahoo.com"; // Your SMTP server. For example, smtp.mail.yahoo.com
                            $username       = "your.email@yahoo.com"; //For example, your.email@yahoo.com
                            $password       = "1234567"; // Your password
                            $SMTPSecure     = "ssl"; // For example, ssl or tls
                            $port           = 465; // For example, 465

    After that you need to open file contact-form.js (line ~39) and change this line: 

                            $.post('php/contact_me.php', post_data, function(response){

    to this line:

                            $.post('php/contact_me_smtp.php', post_data, function(response){

    You can get the email address, host, username and password in admin panel of your web hosting provider.

    Newsletter Form

    To configure the newsletter form, open file /js/jquery.ajaxchimp.min.js and replace Mailchimp form action URL with your Mailchimp list url (~line 9): 

                            url: "http://bestlooker.us10.list-manage.com/subscribe/post?u=e16312d09b1b43f5c26fc526c&id=195aaab5cb" //Replace this with your own mailchimp post URL.

    Use this instruction to get your own Mailchimp list URL:

    1. Sign into your Mailchimp account.
    2. Create new list.
    3. Go to: "Audience / Manage Audience / Signup Forms / Embedded forms" or "Lists / Your list / Signup Form / Embedded forms".
    4. Find the text field "Copy/paste into your site" and copy "action" attribute of the "form" tag (see screenshot).

    FAQ

    1. How to remove smooth scroll?
      Find and delete the line (in the end of the page):

    2. How to remove appear animation from page elements?
      Find and delete the class appear-animate form <body> tag.

    3. Why lightbox video doesn't work?
      Ajax methods of jQuery are used for lightbox videos and they can work only on the real web server. For the correct operation of all functions, upload your website on a real server.

    4. Why contact form may doesn’t work?
      For the correct operation of contact form, the php support is necessary. Upload your website to a web server with support of PHP.

    5. Why there are no images that were in live demo?
      Envato forbids including demonstration images to template’s files. Please create a support request, and we will send you all the free images used in the Resonance template demo.
